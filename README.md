# Assignment 1 - Computer Fundamentals

## 1. Number System
Computer uses binary number system (0 and 1).
Example:
25 in decimal = 11001 in binary

## 2. CPU Registers
- Program Counter (PC)
- Instruction Register (IR)
- Memory Address Register (MAR)
- Memory Data Register (MDR)
- Accumulator (ACC)

Registers are small and fast memory units inside CPU.

## 3. Input-Process-Output Model
Input → Process → Output

Example:
Input: User enters number
Process: Calculate square
Output: Display square
